# Appify Social

A modern social media application built with **Next.js**, **Node.js**, **Express.js**, and **MongoDB** as part of the AppifyLab Full Stack Engineer Selection Task. The application allows users to authenticate, create posts, interact through likes, comments, and replies, and manage public/private content according to the task requirements.

## Features

- User Authentication
  - Sign Up
  - Login
  - Protected Routes

- Feed
  - Create text posts
  - Upload images
  - Public & Private posts
  - Latest posts first

- Social Interactions
  - Like / Unlike posts
  - Comment on posts
  - Reply to comments
  - Like / Unlike comments and replies
  - View users who liked a post/comment/reply

## Getting Started

### Clone Repository

```bash
git clone https://github.com/Md-Rubel-Ahmed-Rana/appify-social.git
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

### Environment Variables

Create a `.env` file inside both the frontend and backend directories.

> Environment variables will be documented after the project is completed.

### Run Development Server

Frontend

```bash
npm run dev
```

Backend

```bash
npm run dev
```
